# SAP BTP Gemini AI Assistant

A Python-based AI middleware deployed on **SAP Business Technology Platform (Cloud Foundry)**.

## 🚀 Features
- Integrates **Google Gemini 2.5 Flash** for real-time generative AI responses.
- Hosted on **SAP BTP** infrastructure.
- Uses environment variables for secure API key management.
- Dynamic query handling via URL parameters.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** Flask
- **Cloud:** SAP Business Technology Platform
- **AI Model:** Google Gemini 2.5

## 📖 How to Use
Access the live endpoint:
`https://my-first-sap-app.cfapps.us10-001.hana.ondemand.com/ask?q=YourQuestionHere`
